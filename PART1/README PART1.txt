Pengenalan HTML Part 1 => Definisi, Penjelasan Struktur, Hello World, Heading, paragraph

Apa itu HTML ?

HTML singkatan dari HyperText Markup Language, ini adalah struktur dasar dari halaman web.
HTML merupakan struktur dari isi konten halaman Web
HTML membutuhkan CSS untuk memperindah tampilan halaman web dan juga JS atau javascript untuk membuat kesan halaman dinamis dan interaktif.

Heading
Heading dalam HTML adalah tag yang digunakan untuk mendefinisikan judul dan subjudul pada sebuah halaman web. Heading tag memiliki urutan nomor dari H1 hingga H6, dengan H1 sebagai judul utama dan H6 sebagai subjudul paling rendah

paragraph
Dalam HTML, paragraf adalah blok teks yang dipisahkan dari blok lain oleh spasi vertikal dan/atau indentasi baris pertama. Paragraf merupakan salah satu elemen HTML yang mendasar dan digunakan untuk mengatur konten tertulis di halaman web. 
Untuk mendefinisikan paragraf dalam HTML, digunakan tag HTML <p>. Tag pembuka <p> digunakan untuk memulai paragraf, sedangkan tag penutup </p> digunakan untuk mengakhirinya.
